## 1. Identitas Proyek

### Nama Project
ZOOTOPIA

### Deskripsi Singkat
Zootopia adalah platform edukasi yang didedikasikan untuk mengenalkan berbagai habitat hewan di seluruh dunia, mulai dari wilayah tropis yang rimbun hingga kutub yang membeku.

Melalui website ini, pengunjung dapat mempelajari karakteristik setiap habitat, hewan-hewan yang hidup di dalamnya, serta pentingnya menjaga kelestarian lingkungan agar keanekaragaman hayati tetap terjaga.

## 2. Teknologi yang Digunakan

### Framework Utama
**BootStrap 5**

**Alasan Penggunaan:**
- memudahkan pengembangan tampilan website yang responsif dan konsisten di  berbagai ukuran layar
- Bootstrap menyediakan grid system, komponen UI siap pakai (seperti navbar, card, dan button)
- utilitas CSS yang mempercepat proses pengembangan tanpa harus menulis kode dari nol

### Teknologi Pendukung
- HTML
- CSS
- SCRIPT-JS

## 3. Daftar Fitur

1. **Navbar**  
   - Navbar (Navigation Bar) berfungsi sebagai menu navigasi utama yang memudahkan pengguna berpindah antar halaman dalam website FishDepth.

2. **Search**  
   - Fitur Search (Pencarian) memungkinkan pengguna mencari spesies laut dengan cepat berdasarkan nama ikan, dan kedalaman laut.

3. **Komentar**  
   - Fitur Komentar memungkinkan pengguna untuk memberikan tanggapan, pertanyaan, atau tambahan informasi terkait spesies laut yang ditampilkan.

4. **Contact Whatsapp**
   - Fitur Contact WhatsApp memungkinkan pengguna menghubungi pengelola website  FishDepth secara langsung melalui WhatsApp dengan satu kali klik.
